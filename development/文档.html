# 本地开发

先决条件：

  • nodejs(Version >= 16)
  • yarn(Version >= 1)

Clone 本仓库

在项目文件夹下运行：

yarn install
yarn docs:dev

yarn add @vuepress/plugin-search@next -d
在 GitHub 上编辑此页 在新窗口打开
